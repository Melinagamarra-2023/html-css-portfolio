🌿 Landing Page - Misiones

Este proyecto es una landing page turística sobre la provincia de Misiones, desarrollada con HTML, CSS y JavaScript.
Fue creada como práctica de desarrollo frontend y como parte de mi portafolio.

🚀 Demo

👉 Puedes ver el proyecto online aquí:
(https://melinagamarra-2023.github.io/html-css-portfolio/)


🛠️ Tecnologías utilizadas

HTML5

CSS3 (Flexbox, Grid, Responsive Design)

JavaScript (menú hamburguesa interactivo)


📱 Características

Diseño responsive (adaptado a celulares)

Menú hamburguesa

Secciones:

Inicio

Historia

Turismo

Contacto

Efecto parallax

Galería de imágenes con Grid

Botón flotante de WhatsApp
